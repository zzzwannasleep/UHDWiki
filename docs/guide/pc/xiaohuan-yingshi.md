# 小幻影视（PC 端）

本页介绍如何在 Windows 上安装并配置 **小幻影视**（以 **Emby** 为例）。

::: tip 准备
- Windows 10 / 11
- Microsoft Store 可正常打开
- 已准备好 UHD 服务器信息（账密 / 播放地址等）
:::

## 安装（Microsoft Store）

### 1）打开 Microsoft Store

<img class="screenshot" loading="lazy" alt="打开 Microsoft Store" src="https://github.com/user-attachments/assets/f09e274d-60e7-4e55-9349-6d1a390b4b16" />

### 2）搜索「小幻影视」

在搜索框输入 **小幻影视**，然后搜索。

<img class="screenshot" loading="lazy" alt="搜索 小幻影视" src="https://github.com/user-attachments/assets/54ac15b0-cb95-4561-9cde-0bb1ca1f69de" />

### 3）下载并安装

找到 **小幻影视** 后，点击下载/安装。

<img class="screenshot" loading="lazy" alt="下载 小幻影视" src="https://github.com/user-attachments/assets/ad8fae8a-09fa-4dd6-92b6-df43cda602c7" />

## 第一次打开与配置（Emby）

### 1）启动应用并选择 Emby

安装完成后打开应用，点击 **Emby**。

<img class="screenshot" loading="lazy" alt="启动后选择 Emby" src="https://github.com/user-attachments/assets/97f15e5b-7b78-4746-a5b3-5e6126cfbcef" />

### 2）填写服务器信息

按页面提示填写服务器地址与账号信息。

<img class="screenshot" loading="lazy" alt="填写 Emby 服务器信息" src="https://github.com/user-attachments/assets/3113a099-4e1e-439b-b6f4-e926b29adeaf" />

::: details 常见需要填写的内容（示例）
- 服务器地址：`https://your-emby-domain.com`
- 端口：`80`（HTTP）或 `443`（HTTPS）
- 账号/密码：你的 Emby 账号
:::

## 检查是否配置成功

- 能看到媒体库/海报墙
- 能正常播放（可快进/切换清晰度）

## 常见问题

### Microsoft Store 搜不到「小幻影视」

- 检查系统地区/语言设置、商店是否可用
- 试试直接在商店网页搜索并跳转到商店安装

### 连接服务器失败 / 一直加载

- 确认服务器地址在浏览器里能打开（同一网络环境下）
- 如果是域名，检查 DNS 或代理/VPN 设置

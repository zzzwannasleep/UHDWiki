# Hills Lite（PC 端）

本页介绍如何在 Windows 上安装并配置 **Hills Lite**。

::: tip 准备
- Windows 10 / 11
- Microsoft Store 可正常打开
- 已准备好 UHD 服务器信息（账密 / 获取到的播放链接）
:::

## 安装（Microsoft Store）

### 1）打开 Microsoft Store

<img class="screenshot" loading="lazy" alt="打开 Microsoft Store" src="https://github.com/user-attachments/assets/be774a82-2ea9-4253-8b9f-4fed4fc0d227" />

### 2）搜索「Hills Lite」

在搜索框输入 **Hills Lite**，然后搜索。

<img class="screenshot" loading="lazy" alt="搜索 Hills Lite" src="https://github.com/user-attachments/assets/33f4528b-bd87-438d-b665-3c5fd66d1e5c" />

### 3）下载并安装

找到 **Hills Lite** 后，点击安装。

<img class="screenshot" loading="lazy" alt="安装 Hills Lite" src="https://github.com/user-attachments/assets/c69cf81a-5ae6-4e0d-b520-5c4caddc0b7f" />

## 第一次打开与添加服务器

### 1）进入「添加服务器」

打开 Hills Lite，点击 **添加服务器**。

<img class="screenshot" loading="lazy" alt="点击 添加服务器" src="https://github.com/user-attachments/assets/d66f4c58-66d6-4330-be78-7813bef8104d" />

### 2）填写信息并登录

按页面提示填写服务器地址与账号信息，点击 **登录**。

<img class="screenshot" loading="lazy" alt="填写服务器信息并登录" src="https://github.com/user-attachments/assets/cd86e04f-c262-466d-ac94-ff6c333fd40b" />

::: details 常见需要填写的内容（示例）
- 服务器地址：`https://your-server-domain.com`
- 端口：按你的服务器配置填写
- 账号/密码：你的账号
:::

## 检查是否配置成功

- 能看到媒体库/海报墙
- 能正常播放（可切换清晰度/字幕等）

## 常见问题

### 搜不到应用 / 无法安装

- 检查 Microsoft Store 是否登录、地区设置是否正常
- 尝试在商店网页搜索后跳转安装

### 登录失败 / 连接超时

- 确认服务器地址可访问（浏览器能打开）
- 检查代理/VPN、DNS 解析是否影响连接

## 完成

配置完成后，就可以开始观影了。

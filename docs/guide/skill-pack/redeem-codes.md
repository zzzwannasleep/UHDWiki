# 兑换码的创建与使用

本页介绍兑换码（优惠/权益码）的 **创建** 与 **使用**（不同角色权限可能不同）。

## 使用兑换码（用户）

1. 进入站点后台（个人中心/控制台）
2. 找到 **兑换码** 入口
3. 粘贴兑换码并提交
4. 刷新查看权益/套餐是否生效

## 创建兑换码（管理员/运营）

::: warning 提示
如果你在后台看不到「创建」按钮，说明你没有相关权限。

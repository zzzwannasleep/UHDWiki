# 视频偏色

「偏色」最常见于 **HDR 片源** 在不支持 HDR 的设备/播放器上播放，或输出色彩空间不匹配导致的色彩异常（发灰、发绿、发紫、肤色不自然等）。

## 常见原因

- **HDR/杜比视界与设备不兼容**：设备不支持或支持不完整
- **HDR → SDR 色调映射（Tone Mapping）不正确**：导致画面发灰或饱和度异常
- **播放器输出设置不匹配**：颜色空间/动态范围设置与显示器/电视不一致
- **线材/接口/电视设置问题**：例如 HDMI 模式、增强模式未开启或开启不当（电视端更常见）

## 处理建议（推荐顺序）

1. **先切换到 SDR 版本**（如果同一影片同时提供 SDR/HDR）
2. **关闭“强制 HDR/杜比视界”** 之类的选项（若播放器/系统有）
3. **更新播放器与系统**：不少偏色问题来自解码与色调映射的兼容性
4. **检查显示设备设置**：电视/显示器的画面模式、HDR 模式、HDMI 增强模式等
5. **换播放器/换设备验证**：判断是资源问题还是本机兼容问题

## 你可以如何描述问题（便于排查）

- 哪个设备（手机/电视/电脑）出现偏色，其他设备是否正常
- 影片规格（是否 HDR/杜比）
- 具体表现（发灰/发绿/发紫/过饱和）
- 是否切到 SDR 后恢复正常

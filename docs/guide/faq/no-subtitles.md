# 剧集没字幕

「没字幕」通常有三类情况：**资源本身不带字幕**、**字幕轨道没选对/被关闭**、或 **播放器不支持某些字幕格式**。

## 先确认：字幕有没有被关闭

- 播放器里找到 **字幕/CC** 菜单
- 确认字幕不是「关闭」
- 尝试切换不同字幕轨（中/英、简/繁等）

## 常见原因

- **资源不包含字幕**：有些版本只带音轨不带字幕
- **默认字幕语言不匹配**：需要手动选择字幕轨
- **字幕格式不兼容**：部分播放器对 PGS/ASS 支持有限
- **转码/直通策略导致字幕不可用**：某些组合下字幕需要转码或烧录

## 处理建议

1. **换一个版本/片源**：优先选择标注有字幕的版本
2. **在播放器里手动切字幕轨**：不要只看默认选择
3. **换客户端验证**：判断是字幕格式不兼容还是资源缺失
4. **反馈时提供信息**：片名、集数、你看到的字幕轨列表截图（若有）

::: tip 小提示
如果你看到有字幕轨但选了仍不显示，优先怀疑「播放器对该字幕格式不支持」或「输出渲染异常」。
:::

# 重置流量

本页说明如何在账号后台进行 **流量/配额重置**（具体规则以站点页面提示为准）。

::: tip 什么时候需要重置？
- 套餐流量用尽，无法继续播放/下载
- 想把流量恢复到可用状态（按规则允许的情况下）
:::

## 注意事项

- 有些套餐是 **按周期自动重置**（例如按月），不需要手动操作
- 部分重置可能会 **扣除权益/次数**，请先看清提示再确认
- 重置后如仍不可用，建议先退出登录再重新登录

## 操作步骤

1. 打开站点后台（个人中心/控制台）
2. 进入 **套餐/流量/账户** 相关页面
3. 找到 **重置流量**（或类似名称）按钮
4. 阅读提示信息，确认后提交
5. 刷新页面确认剩余流量是否已更新

::: details 找不到「重置流量」按钮？
- 可能当前套餐不支持手动重置
- 可能你没有可用的重置次数/权益
- 也可能入口在「订单/套餐详情」里
:::

## 常见问题

### 重置后仍然提示无流量

- 刷新页面或重新登录
- 等待 1–2 分钟让状态同步
- 确认播放设备使用的是同一个账号

### 自动重置和手动重置有什么区别？

- 自动重置：按周期系统自动恢复（通常不消耗额外权益）
- 手动重置：在规则允许时手动触发（可能消耗次数/权益）
